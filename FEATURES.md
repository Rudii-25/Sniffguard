# ✨ Advanced Features

**SniffGu@rd V2** is packed with powerful features designed for wireless security professionals and enthusiasts.

## 📡 Network Capabilities

### 🔍 Real-Time Scanning
- **Live Detection**: Discover all nearby access points in real-time.
- **Detailed Intel**: Get SSID, BSSID, Signal Strength (RSSI), Channel, and Encryption Type.
- **Band Detection**: Supports both 2.4GHz and 5GHz (Dual Band) scanning.

### 🛡️ Security Analysis
- **Unsecured Network Alerts**: Instantly identify open or weak WEP networks.
- **Hidden SSID Discovery**: Specialized techniques to reveal hidden network names.

## 🛠️ Technical Highlights

| Feature | Description | Status |

## 📈 Dashboard Overview

The GUI provides a centralized command center where you can:
- **Monitor** all wireless traffic in a clean table view.
- **Visualize** signal strength to locate access points physically.
- **Manage** capture files and exported logs effortlessly.
