# 🤝 Contributing to SniffGu@rd

We love your input! We want to make contributing to SniffGu@rd as easy and transparent as possible.

## 🛣️ How Can You Help?

### 🐛 Reporting Bugs
- Use the [GitHub Issue Tracker](https://github.com/Rudii-25/Sniffguard/issues).
- Describe the bug clearly and provide steps to reproduce.
- Include OS details and your WiFi adapter model.

### 💡 Feature Requests
- Have an idea? [Open an issue](https://github.com/Rudii-25/Sniffguard/issues) and label it as a `feature request`.
- Explain why the feature would be useful for the community.

### 👩‍💻 Pull Requests
1. Fork the repo on [GitHub](https://github.com/Rudii-25/Sniffguard).
2. Create a new branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. [Open a Pull Request](https://github.com/Rudii-25/Sniffguard/pulls).

---

## 📜 Code of Conduct
Please be respectful and professional in all interactions within this project's ecosystem.
