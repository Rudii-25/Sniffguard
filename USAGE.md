# 📖 Usage Manual

Master the power of **SniffGu@rd V2** with this comprehensive usage guide.

## 🎮 Interface Navigation

### 1. Interface Selection
Upon launching, click on **"Detect Interfaces"**. The tool will scan your system for wireless cards (e.g., `wlan0`, `wlan1`). Select the one you wish to use.

### 2. Starting the Scan
Click the **"Start Scan"** button. SniffGu@rd will put your card into Monitor Mode (if supported) and begin hopping through channels to find networks.

### 3. Monitoring Results
The main dashboard table will populate with:
- **SSID**: The name of the WiFi network.
- **BSSID**: The MAC address of the Access Point.
- **PWR**: Signal strength (closer to 0 is stronger).
- **CH**: The channel the network is operating on.

## ⚙️ Advanced Operations

### 📁 Exporting Data
You can save your scan results for later analysis. Use the `File -> Export` menu to save as CSV or JSON.

### 📝 Viewing Logs
Detailed activity logs are stored in the `logs/` directory. These are useful for debugging or forensic analysis.

---

> [!TIP]
> **Better Results**: Use an external high-gain WiFi adapter like Alfa Network for significantly better range and sensitivity.
